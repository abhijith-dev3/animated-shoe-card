#Nike Shoe UI Card

[DEMO]()


A modern and interactive nike product card UI built with HTML, CSS AND JS

##FEATURES
Animated shoe image that falls and rotates when changin colors.
Interactive color section selection buttons that update the shoe image dynamically.
Stylish card with gradient backgrounds and responsive design.
Add-to-cart button with subtle animation for user feedback.
Fully responsive layout for mobile and desktop screen.

##TECHNOLOGIES USED
HTML5
CSS3(Flexbox, Grid, Animations, Gradients)
JavaScript(DOM manipulation, event handling)
